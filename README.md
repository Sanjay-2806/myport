# Minimalist Portfolio Landing Page

A ultra-minimalist portfolio landing page with a monochrome black-and-white design featuring a Yin Yang symbol as the central focal point.

## Features

- Pure white background with high contrast black elements
- Large Yin Yang symbol as the central element
- Clean, modern sans-serif typography
- Vertical navigation elements on left and right sides
- Bottom center navigation
- Social media icons column
- Responsive design

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Design Elements

- **Top Left**: Cursive "sanjay" logo
- **Top Right**: "Say hi.." text
- **Left Navigation**: "Experience" and "Projects" (vertical)
- **Right Navigation**: "Feats" (vertical)
- **Center**: Yin Yang symbol with "click here" text below
- **Bottom Center**: "About" and "My Skills" navigation
- **Left Bottom**: Social media icons (LinkedIn, GitHub, Twitter, Facebook, YouTube)

## Technology Stack

- React 18
- Vite
- CSS3
- Modern web fonts (Inter, Dancing Script)
